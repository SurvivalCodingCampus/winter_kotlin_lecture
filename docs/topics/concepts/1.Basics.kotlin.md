## 타입체크 자동 케스팅 
```kotlin
// is operator를 사용하면 타입을 체크해서 return이 자동으로 casting된다.
fun getStringLength(obj: Any): Int?{
   if( obj is String) {
       return obj.length
   }
    return null
}

// 아래있는게 더 깔끔해보인다.
fun getStringLength(obj: Any): Int? {
    if (obj !is String) return null

    // `obj` is automatically cast to `String` in this branch
    return obj.length
}
```

- 타입이 굉장히 쉽게 구현이 되는거 같다.
